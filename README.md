# QR Code Component

A simple and responsive **QR Code Component** built using **HTML and CSS**. This project is part of a Frontend Mentor challenge and focuses on layout, typography, and responsive design fundamentals.

---

## Preview

![QR Code Component Preview](./images/image-qr-code.png)

---

## Built With

- Semantic HTML5
- CSS
  - CSS Grid
  - CSS Variables
  - Google Font (Outfit)
- Mobile-first workflow

---

## Features

- Responsive card layout
- Centered content using CSS Grid
- Clean and minimal UI
- Custom color palette with CSS variables
- Accessible image and text structure

## What I Learned

- How to center content vertically and horizontally using CSS Grid
- Better use of spacing and typography
- Writing clean and maintainable CSS
- Using CSS custom properties for consistent theming
